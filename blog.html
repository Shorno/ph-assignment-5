Important FAQ

What is the Document Object Model (DOM)

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, allowing programs to manipulate the document's structure, style, and content. The DOM provides a way for scripts to update the content, structure, and style of a document while it is being viewed.

How do you select an element from DOM?

getElementById: Selects an element by its ID.
getElementsByClassName: Selects elements by their class name.
getElementsByTagName: Selects elements by their tag name.
querySelector: Selects the first element that matches a CSS selector.
querySelectorAll: Selects all elements that match a CSS selector.

What is event delegation in the context of DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to manage events for multiple child elements. Instead of adding individual event listeners to each child element, the parent element's event listener can handle events from its children using event propagation (bubbling).

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM, you can use various methods and properties provided by the DOM API. The getAttribute method retrieves the value of an attribute, while the setAttribute method sets the value of an attribute, and the removeAttribute method removes an attribute. For manipulating styles, you can use the style property to directly set the CSS properties of an element. For example, you can change the text color, background color, and font size of an element by setting the color, backgroundColor, and fontSize properties, respectively.
